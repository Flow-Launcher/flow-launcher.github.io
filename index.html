Flow Launcher

Quick File Search & App Launcher for Windows

Sponsored by

Start Flow Launcher anytime, anywhere.

Use the hotkey what you want. A search window is displayed immediately.

※ Default hotkey is Alt+Space

Features

Open Apps

Search for apps.

Search Files

Files or file contents by windows search indexer / everything.

Web Searches & Urls

Quick googling or open website in your desktop.

Search Bookmarks

Search Browser's bookmarks

System Commands

Restart, Reboot, Shutdown, Etc...

Shell Commands

Run batch and PowerShell commands as Administrator or a different user

Calculator

Do mathematical calculations and copy the result to clipboard.

Windows Settings

Search for Windows & Control Panel settings.

Plugin Store

You can view the full plugin list or quickly install a plugin via the Plugin Store menu inside Settings.

Customazations

Support Themes, Animation & Sounds, Dark Scheme.
